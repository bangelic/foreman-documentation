= Searching RHEL Lifecycle Status

You can use the *Search* field to search hosts by `rhel_lifecycle_status`.
It can have the following values:

* `full_support`
* `maintenance_support`
* `approaching_end_of_maintenance`
* `extended_support`
* `approaching_end_of_support`
* `support_ended`
